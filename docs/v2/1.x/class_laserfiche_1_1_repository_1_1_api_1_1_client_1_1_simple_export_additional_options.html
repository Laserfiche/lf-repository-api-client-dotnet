◆ ExportImagePagePrefix

string ExportImagePagePrefix = ", Page "
getset

The page prefix of the individual files, when exporting to multi-file format (e.g.zip) when part=Pages.
The value must have a length of atmost 10 characters.
Further, as the value is used in the file name, it cannot include characters that are invalid from the point of view of the file
These invalid characters are: colon (:), slash, backslash, double-quotation, asterisk, question mark, Angle brackets(i.e. less than, greater than), and vertical bar (|)
An empty string or all-whitespace string is allowed.
